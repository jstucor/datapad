# Star Wars x Warhammer
I love the gameplay of Warhammer, but have a bigger passion for Star Wars. With both of these loves combined, I wanted to make this complex game a bit more simple for those who have never played it and smooth for those who do play it.
With this I present the datapad. The concept is simple, instead of flipping through pdfs to find the stats of a character, simply tap it to the datapad to get all the info of the character and return it to the board.


## Implementing your own
Before you decide to invest in your own hardware and whatnot, I would begin by testing this out with your computer by switching the ARDUINO variable to false. From there you can test everything in your CLI without spending a dime. 
Once you have purchased your hardware, the setup is quite simple, push the CPP code to your Arduino Nano and run the python code on your pi.
